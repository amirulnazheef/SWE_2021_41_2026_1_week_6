# SWE_2021_41_2026_1_week_6

# Overview
This assignment focuses on **Bash scripting** to automate file organization.  
The script arranges files into directories (`a` to `z`) based on the **first letter of each filename**.

# Workflow
- All files are initially stored inside the `files/` directory  
- The script:
  1. Reads each file in `files/`
  2. Extracts the first character of the filename
  3. Converts it to lowercase
  4. Moves the file into the corresponding folder
